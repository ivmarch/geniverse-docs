# Система Дизайну

Система дизайну GeniVerse надає згуртовану візуальну мову та бібліотеку компонентів, яка забезпечує послідовність, доступність та сучасний користувацький досвід на всіх платформах та інтерфейсах.

## Дизайн Токени

### Кольорова Палітра

**Основні Кольори**
- **Фон**: `#0B0D0C` - Глибокий темний фон для зменшення напруги очей
- **Поверхня**: `#141716` - Підняті поверхні та картки
- **Акцент**: `#34E1A1` - Основний акцентний колір для CTA та виділень
- **Текст Основний**: `#E8F9F0` - Основний колір тексту для високого контрасту
- **Текст Вторинний**: `#B8D4C5` - Вторинний текст для меншого акценту

**Семантичні Кольори**
- **Успіх**: `#34E1A1` - Стани успіху та позитивний зворотний зв'язок
- **Попередження**: `#FFB84D` - Попередження та обережні повідомлення
- **Помилка**: `#FF6B6B` - Помилки та критичні сповіщення
- **Інформація**: `#4DABF7` - Інформаційні повідомлення

**Нейтральні Кольори**
- **Межа**: `#1F2321` - Межі та роздільники
- **Наведення**: `rgba(52, 225, 161, 0.1)` - Стани наведення
- **Активний**: `rgba(52, 225, 161, 0.15)` - Активні стани
- **Накладання**: `rgba(11, 13, 12, 0.9)` - Модальні та накладання фонів

### Типографіка

**Сімейства Шрифтів**
- **Заголовки**: `'Garet'` - Сучасний геометричний sans-serif для заголовків
- **Основний Текст**: `'Inter'` - Чистий, читабельний sans-serif для основного тексту
- **Моноширинний**: `'JetBrains Mono'` - Код та технічний контент

**Шкала Типів**
- **H1**: `2.5rem` / `3rem` висота рядка - Заголовки сторінок
- **H2**: `2rem` / `2.5rem` висота рядка - Заголовки розділів
- **H3**: `1.5rem` / `2rem` висота рядка - Заголовки підрозділів
- **H4**: `1.25rem` / `1.75rem` висота рядка - Менші заголовки
- **Основний Великий**: `1.125rem` / `1.75rem` висота рядка - Виділений основний текст
- **Основний**: `1rem` / `1.5rem` висота рядка - Стандартний основний текст
- **Основний Маленький**: `0.875rem` / `1.25rem` висота рядка - Вторинний текст
- **Підпис**: `0.75rem` / `1rem` висота рядка - Підписи та мітки

**Ваги Шрифтів**
- **Легкий**: `300` - Тонкий акцент
- **Звичайний**: `400` - Стандартний текст
- **Середній**: `500` - Середній акцент
- **Напівжирний**: `600` - Сильний акцент
- **Жирний**: `700` - Максимальний акцент

### Відступи

**Шкала Відступів** (базова одиниця 8px)
- `4px` - XS відступ
- `8px` - S відступ
- `16px` - M відступ
- `24px` - L відступ
- `32px` - XL відступ
- `48px` - 2XL відступ
- `64px` - 3XL відступ
- `96px` - 4XL відступ

### Радіус Меж

- `4px` - Малі елементи (значки, теги)
- `6px` - Стандартні елементи (кнопки, поля введення)
- `8px` - Картки та контейнери
- `12px` - Великі контейнери
- `16px` - Модальні та діалоги

### Тіні

- **Мала**: `0 1px 2px rgba(0, 0, 0, 0.1)`
- **Середня**: `0 4px 8px rgba(0, 0, 0, 0.15)`
- **Велика**: `0 8px 16px rgba(0, 0, 0, 0.2)`
- **Акцент**: `0 4px 12px rgba(52, 225, 161, 0.2)`

## Бібліотека Компонентів

### Кнопки

**Основна Кнопка**
- Фон: Акцентний колір
- Текст: Темний колір фону
- Наведення: Темніший відтінок акценту
- Активний: Натиснутий стан з тінню

**Вторинна Кнопка**
- Фон: Прозорий
- Межа: Акцентний колір
- Текст: Акцентний колір
- Наведення: Акцентний фон з низькою непрозорістю

**Третинна Кнопка**
- Фон: Прозорий
- Текст: Основний колір тексту
- Наведення: Фон поверхні

### Форми

**Поля Введення**
- Фон: Колір поверхні
- Межа: Колір межі
- Текст: Основний колір тексту
- Фокус: Акцентна межа та свічення
- Плейсхолдер: Вторинний колір тексту

**Мітки**
- Шрифт: Основний маленький, напівжирний
- Колір: Основний колір тексту
- Відступ: 8px під міткою

**Стани Помилок**
- Межа: Колір помилки
- Текст: Колір помилки
- Фон: Колір помилки з низькою непрозорістю

### Картки

**Стандартна Картка**
- Фон: Колір поверхні
- Межа: Колір межі
- Радіус межі: 8px
- Відступ: 24px
- Тінь: Середня тінь

**Інтерактивна Картка**
- Наведення: Акцентна межа
- Тінь: Акцентна тінь при наведенні
- Перехід: Плавні переходи

### Навігація

**Навбар**
- Фон: Колір фону з межею
- Висота: 64px
- Межа: Нижня межа з кольором межі

**Бічна Панель**
- Фон: Колір поверхні
- Межа: Права межа з кольором межі
- Ширина: 280px (згорнута: 64px)

**Елементи Меню**
- Відступ: 12px 16px
- Радіус межі: 6px
- Наведення: Акцентний фон з низькою непрозорістю
- Активний: Акцентний фон, акцентний текст

### Компоненти Типографіки

**Заголовки**
- Сімейство шрифтів: Garet
- Вага шрифту: Напівжирний
- Міжлітерний інтервал: -0.02em
- Колір: Основний колір тексту

**Основний Текст**
- Сімейство шрифтів: Inter
- Вага шрифту: Звичайний
- Висота рядка: 1.5
- Колір: Основний колір тексту

**Посилання**
- Колір: Акцентний колір
- Наведення: Світліший відтінок акценту
- Підкреслення: При наведенні (опціонально)

### Блоки Коду

**Підсвічування Синтаксису**
- Фон: Колір поверхні
- Межа: Колір межі
- Шрифт: Моноширинний шрифт
- Тема: Темна тема з акцентними виділеннями

### Таблиці

**Структура Таблиці**
- Заголовок: Фон поверхні
- Рядки: Альтернативний фон поверхні
- Межа: Колір межі
- Наведення: Акцентний фон з низькою непрозорістю

## Патерни Макету

### Система Сітки

**12-Колонкова Сітка**
- Максимальна ширина: 1280px
- Проміжок: 24px
- Точки Перелому:
  - Мобільний: < 768px
  - Планшет: 768px - 1024px
  - Десктоп: > 1024px

### Розміри Контейнерів

- **Малий**: Максимальна ширина 640px
- **Середній**: Максимальна ширина 960px
- **Великий**: Максимальна ширина 1280px
- **Повний**: Ширина 100%

### Патерни Відступів

- **Відступ Розділу**: 64px вертикально
- **Відступ Компонента**: 32px вертикально
- **Відступ Елемента**: 16px вертикально
- **Внутрішній Відступ**: 8px горизонтально

## Доступність

### Контраст Кольорів

- **Текст на Фоні**: Мінімальне співвідношення 4.5:1 (WCAG AA)
- **Великий Текст**: Мінімальне співвідношення 3:1
- **Інтерактивні Елементи**: Мінімальне співвідношення 3:1

### Стани Фокусу

- **Кільце Фокусу**: 2px суцільний акцентний колір
- **Зміщення Фокусу**: 2px зміщення контуру
- **Навігація з Клавіатури**: Всі інтерактивні елементи доступні

### Екранові Читачі

- **Семантичний HTML**: Правильні HTML елементи
- **ARIA Мітки**: Описові мітки для складних компонентів
- **Альтернативний Текст**: Всі зображення мають описовий альтернативний текст
- **Орієнтири**: Правильне використання орієнтирів

## Адаптивний Дизайн

### Мобільний Перший

Дизайн починається з мобільного та розширюється для більших екранів.

### Брейкпоінти

- **Мобільний**: До 768px
- **Планшет**: 768px - 1024px
- **Десктоп**: Понад 1024px

### Адаптивні Компоненти

Компоненти адаптуються до розміру екрану:
- Навігація перетворюється на гамбургер меню на мобільних
- Таблиці стають прокручуваними або картковими на малих екранах
- Форми оптимізуються для мобільних пристроїв

## Темна Тема

### Підтримка Темної Теми

Всі компоненти підтримують темну тему з правильними контрастами та кольорами.

### Перемикання Тем

- Автоматичне виявлення переваг системи
- Ручне перемикання користувачем
- Збереження вибору користувача

## Анімації та Переходи

### Принципи Анімації

- **Тривалість**: Короткі анімації (200-300ms)
- **Easing**: Природні криві easing
- **Мета**: Анімації покращують UX, не відволікають

### Типи Анімацій

- **Переходи**: Плавні переходи між станами
- **Завантаження**: Індикатори прогресу та скелетони
- **Взаємодії**: Відгуки на дії користувача
- **Мікро-взаємодії**: Тонкі анімації для покращення досвіду
