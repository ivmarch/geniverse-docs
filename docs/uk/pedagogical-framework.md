# Педагогічна Рамка

Педагогічна рамка GeniVerse інтегрує множинні теорії навчання та методології для створення ефективних, захоплюючих освітніх досвідів. Рамка керує створенням контенту, персоналізацією ШІ та дизайном навчальних шляхів.

## Теорії Навчання

### Конструктивізм

Учні активно конструюють знання через досвід та рефлексію. GeniVerse підтримує конструктивістське навчання через:

- Надання імерсивних, практичних досвідів
- Заохочення дослідження та експериментування
- Сприяння рефлексії та синтезу знань
- Підтримку спільного побудови знань

### Теорія Соціального Навчання

Навчання відбувається через спостереження, імітацію та соціальну взаємодію. Платформа уможливлює соціальне навчання через:

- Багатокористувацькі XR середовища
- Інструменти співпраці з однолітками
- Функції наставництва та коучингу
- Спільно-орієнтований контент та обговорення

### Теорія Когнітивного Навантаження

Навчання оптимізується, коли когнітивні ресурси керуються ефективно. GeniVerse застосовує принципи когнітивного навантаження через:

- Розбиття інформації відповідно
- Надання підтримки, яка зникає з розвитком експертизи
- Мінімізацію зайвого когнітивного навантаження
- Оптимізацію внутрішнього навантаження через адаптивну складність

### Адаптивне Навчання

Контент та темп адаптуються до індивідуальних потреб учнів. Шар ШІ GeniVerse уможливлює:

- Корекцію складності в реальному часі
- Персоналізовану послідовність контенту
- Шляхи виправлення та збагачення
- Прогрес на основі майстерності

### 1. Оцінити

**Постійна оцінка стану учня**

- Попередня оцінка для встановлення базових знань
- Формативні оцінки під час навчання
- Підсумкові оцінки для перевірки майстерності
- Метрики поведінки та залучення
- Виявлення стилю навчання та переваг

### 2. Адаптувати

**Персоналізація на основі ШІ**

- Корекція складності контенту
- Оптимізація навчального шляху
- Вибір модальності (візуальна, слухова, кінестетична)
- Адаптація темпу
- Маршрутизація виправлення або збагачення

### 3. Доставити

**Багатомодальна доставка контенту**

- Імерсивні XR досвіди
- Інтерактивні симуляції
- Відео та аудіо контент
- Текстовий матеріал
- Спільні діяльності

### 4. Залучити

**Активне навчання та взаємодія**

- Елементи гейміфікації
- Негайний зворотний зв'язок
- Візуалізація прогресу
- Можливості соціального навчання
- Визнання досягнень

### 5. Виміряти

**Комплексна навчальна аналітика**

- Вимірювання приросту знань
- Відстеження розвитку навичок
- Метрики залучення
- Аналіз часу на завдання
- Прогнозування результатів
