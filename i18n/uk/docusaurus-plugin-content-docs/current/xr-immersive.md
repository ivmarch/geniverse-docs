# XR та імерсивність

GeniVerse може використовувати технології розширеної реальності (XR) — віртуальну реальність (VR), доповнену реальність (AR) та змішану реальність (MR) — як додатковий інструмент для візуалізації складних тем, практичних симуляцій і навчання через взаємодію.

*XR не є обов'язковою складовою навчального процесу. Платформа проєктується так, щоб повноцінно працювати без XR, а імерсивні сценарії застосовуються там, де вони дають вимірювану педагогічну користь.*

---

## Технології XR

### Віртуальна реальність (VR)

Повністю імерсивні віртуальні середовища для симуляцій і практики.

**Підтримувані можливості:**

- підтримка поширених VR-шоломів (Oculus/Meta, HTC Vive, Pico та інші)
- сценарії кімнатного масштабу для руху та взаємодії
- відстеження рук для взаємодії без контролерів (за наявності підтримки пристрою)
- просторовий звук для орієнтації та реалістичного аудіосупроводу

**Приклади використання:**

- віртуальні екскурсії до історичних місць
- наукові симуляції та експерименти
- тренування медичних процедур
- візуалізація архітектури та інженерних об'єктів
- середовища мовного занурення

---

### Доповнена реальність (AR)

Накладання цифрового контенту на реальний світ для навчання «на місці».

**Підтримувані можливості:**

- мобільні AR-сценарії на смартфонах і планшетах
- підтримка AR-пристроїв за наявності відповідних SDK
- AR на основі маркерів для стабільного розміщення контенту
- AR без маркерів із просторовим розумінням сцени

**Приклади використання:**

- анатомічні накладання на фізичні моделі або навчальні макети
- інтерактивні підручники з 3D-моделями
- розв'язання задач у реальному середовищі з цифровими підказками
- тренування обслуговування та ремонту обладнання
- освітні сценарії для музеїв і виставок

---

### Змішана реальність (MR)

Поєднання реального та віртуального із просторовою прив'язкою об'єктів.

**Підтримувані можливості:**

- просторове закріплення віртуальних об'єктів у реальному просторі
- коректна взаємодія візуальних шарів (оклюзія) за наявності підтримки пристрою
- інтеграція фізики для взаємодії з об'єктами
- спільні MR-сесії для роботи кількох користувачів у одному просторі

**Приклади використання:**

- спільне проєктування та прототипування
- тренування складання та монтажу
- візуалізація архітектурних рішень у реальних просторах

---

## Створення XR-контенту

### Конструктор XR без програмування

Інструменти для створення імерсивних сценаріїв без написання коду:

- візуальний редактор зі створенням сцен через перетягування елементів
- бібліотека асетів: 3D-моделі, середовища, взаємодії
- система поведінок для налаштування взаємодій
- шаблони для типових навчальних XR-сценаріїв

### Управління 3D-контентом

- імпорт 3D-моделей із поширених форматів (glTF, FBX, OBJ)
- автоматична оптимізація моделей для продуктивності
- обробка та стиснення текстур
- підтримка анімації та відтворення 3D-сцен

### Дизайн взаємодій

- захоплення та маніпуляція об'єктами у 3D
- телепортація та інші безпечні методи переміщення у VR
- 3D-елементи інтерфейсу та плаваючі панелі
- жести та події взаємодії з урахуванням підтримки пристрою

---

## Імерсивні навчальні сценарії

### Віртуальні лабораторії

Безпечні та повторювані практичні експерименти:

- фізичні симуляції для демонстрації принципів і процесів
- хімічні лабораторні сценарії з процедурною логікою
- біологічні моделі та анатомічні дослідження
- інженерне прототипування та тестування рішень

### Історичні реконструкції

Навчання через контекст і взаємодію:

- відтворення історичних локацій
- сценарії різних епох і культур
- інтерактивні наративи для занурення в події

### Тренування навичок

Практика в умовах, де помилка не створює ризику:

- тренування медичних процедур
- навчання роботі зі складним обладнанням
- підготовка до надзвичайних ситуацій
- практика художніх і дизайнерських навичок

### Соціальні навчальні простори

Спільна робота в XR-середовищах:

- віртуальні класи та сесії навчання
- групові заняття та робота в командах
- презентації результатів у 3D-середовищах
- міжнародні навчальні обміни та взаємодія

---

## Продуктивність та оптимізація XR

### Орієнтири продуктивності

- частота кадрів для комфортної роботи у VR
- мінімізація затримки взаємодії
- підтримка достатньої якості рендерингу для читабельності та орієнтації
- оптимізація енергоспоживання для автономних пристроїв

### Техніки оптимізації

- рівні деталізації (LOD) для об'єктів на різній відстані
- оклюзійне відсікання для прихованих елементів сцени
- потокове завантаження текстур за потреби
- інстансування геометрії для повторюваних об'єктів

### Адаптація під пристрій

- автоматичне виявлення можливостей пристрою
- адаптивні профілі якості
- режими зниження навантаження для слабших пристроїв
- кросплатформенна узгодженість сценаріїв

---

## Доступність у XR

### Комфорт і безпека

- режими комфорту зі зменшеним рухом
- альтернативні способи переміщення
- налаштування поля зору та параметрів комфорту
- нагадування про перерви

### Опції доступності

- субтитри та текстові підписи для аудіоконтенту
- візуальні індикатори для звукових подій
- спрощені схеми керування
- налаштування масштабу та дистанцій взаємодії

### Інклюзивний дизайн

- можливість використання сидячи
- сценарії для одноручної взаємодії
- голосові команди як альтернативний спосіб керування (за потреби)
- підтримка допоміжних пристроїв у межах доступних інтеграцій

## Підтримка WebXR

### XR у браузері

XR-сценарії, доступні через веббраузер:

- використання WebXR API як стандартного підходу
- доступ без встановлення окремого застосунку
- робота на різних пристроях і браузерах за наявності підтримки
- прогресивне покращення з коректним режимом без XR

### Функції WebXR

- запуск і керування XR-сесіями
- обробка введення: контролери та руки (за підтримки пристрою)
- ефективний рендеринг на базі WebGL
- просторове відстеження позиції голови та рук

## Аналітика та інсайти

### XR-метрики

- показники присутності та залученості
- аналітика взаємодій у сценах
- патерни навігації та руху
- тривалість активностей у XR

### Ефективність навчання

- оцінка переносу навичок у реальні задачі (за методикою курсу)
- порівняння засвоєння матеріалу між XR та іншими форматами
- аналіз залученості у XR у порівнянні з альтернативними модальностями
- оцінка результатів навчання в XR-сценаріях

---

## Інтеграція з платформою

### Узгоджений досвід

- єдиний обліковий запис для всіх модальностей
- синхронізація прогресу між пристроями та форматами
- безперервність навчання під час переходу між платформами
- соціальна взаємодія незалежно від модальності

### Кросмодальне навчання

- використання XR як доповнення до базового контенту
- закріплення понять через практику в симуляціях
- застосування знань у контекстних сценаріях
- поєднання результатів навчання з різних форматів у єдиній аналітиці

---

## Подальший розвиток XR (опційно)

### Технологічні напрями

- тактильний зворотний зв'язок
- відстеження погляду для взаємодії та аналітики
- повніше відстеження тіла за наявності підтримки пристроїв
- експериментальні інтерфейси взаємодії (за доцільності та безпечності)

### Розширення можливостей

- середовища, згенеровані ШІ, для створення навчальних сцен
- підвищення візуальної якості з урахуванням продуктивності
- «постійні світи» для довгострокових навчальних сценаріїв
- узгоджені переходи між AR, VR, MR та навчанням без XR
